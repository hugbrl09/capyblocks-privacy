Política de Privacidade – Capy Blocks

Última atualização: 07/12/2025

1. Informações que Coletamos

O Capy Blocks não coleta, armazena ou solicita dados pessoais como nome, e-mail, telefone ou localização.

2. Informações Técnicas

Bluetooth: o aplicativo utiliza Bluetooth apenas para conectar os bloquinhos físicos do jogo. Nenhum dado identificável do dispositivo é coletado.

Dados de uso: nenhum dado de uso é enviado para servidores externos.

3. Armazenamento de Dados

O aplicativo pode armazenar localmente informações de progresso do jogo. Esses dados permanecem somente no dispositivo do usuário.

4. Compartilhamento de Informações

O Capy Blocks não compartilha dados com terceiros, não usa serviços externos de análise e não exibe anúncios.

5. Segurança

Adotamos boas práticas de segurança. Como não coletamos dados pessoais, o risco é mínimo.

6. Privacidade de Crianças

O aplicativo pode ser utilizado por crianças pois nenhuma informação pessoal é coletada.

7. Alterações

Esta Política de Privacidade pode ser atualizada. A versão mais recente estará sempre disponível neste endereço.
